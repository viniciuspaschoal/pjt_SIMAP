# pjt_SIMAP

O SIMAP é um sistema desenvolvido para melhorar o acompanhamento pedagógico nas escolas municipais de Santa Bárbara d'Oeste, facilitando o monitoramento do desempenho dos alunos e o planejamento pedagógico. O projeto é realizado em parceria com a Secretaria de Educação, utilizando tecnologias como HTML, CSS, JavaScript e React.
